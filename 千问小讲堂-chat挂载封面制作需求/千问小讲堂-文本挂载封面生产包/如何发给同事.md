# 如何发给同事

## 推荐方式

1. 在 Finder 中右键 **`千问小讲堂-封面生产包`** → **压缩**
2. 得到 `.zip` 后通过飞书/邮件发送（约 **40MB**，主要是字体文件）
3. 让对方解压后先打开 **`README.md`** 或 **`00-从这里开始.md`**

## 包里有什么 / 没有什么

| 包含 | 不包含（需各自本机配置） |
|------|-------------------------|
| 压图脚本 + 字体 | ossutil（OSS 上传工具） |
| 全部说明与 Skill | 你的 CSV 源表（放 `输入表/` 自行添加） |
| 历史 manifest 示例 | 已生成的大图 png（避免包过大） |
| Cursor 话术模板 | mentorx 文案链路全套 Python（太大，见 README） |

## 对方需要安装

- macOS 或 Linux 均可；Windows 需自行试跑 Python 命令
- Python 3：`pip3 install -r 脚本/requirements.txt`

## 发给 Cursor 时

把整个文件夹拖进 Cursor 工作区，或对话里 `@千问小讲堂-封面生产包`，并粘贴 **`发给Cursor的开场话术.md`** 里对应段落。

## 更新本包

由维护人在 **`脚本/`** 更新代码后，重新压缩发送即可。文档变更在 **`文档/logs/项目演进记录.md`** 记一笔。
