## 语文分支

### 内容类型 → content_kind

| 原表类型 | content_kind | 学名示例 |
|----------|--------------|----------|
| 古诗词 | 古诗词 | `古诗词《静夜思》` |
| 文言文 | 文言文 | `文言文《学弈》` |
| 寓言/神话/故事课文 | 寓言课文 / 神话课文 / 故事课文 | `寓言故事《狐狸》` |
| 现代课文 | 现代课文 | `课文《xxx》` |
| 写作 | 写作 | `写作《xxx》` |
| 科普 | 科普 | `科普《xxx》` |
| 妙题 | 妙题文史 / 妙题科普 / 妙题逻辑 | 按子类 |

> **用户可覆盖学名规则**（如成语 tab 只要 `守株待兔` 不要前缀）。

### 创作顺序（强制）

1. **考据** kaoju：朝代、服饰、道具、场景  
2. **三钩子** titles：三公式各一，≤12 字  
3. **画面** hero / scene / ground / mood / footing  
4. **生图** `render_yuwen_prompt` 自动拼铁律  

**脚本**：`yuwen_data_*.py` → `yuwen_batch.py` → `build_yuwen.py` → `validate_yuwen.py`  
**授权规范**：`../scripts/YUWEN_AUTHOR_SPEC.md`

### 红色主题

命中领导人/先烈/革命史/国旗天安门等 → `red_theme` + `red_level`（L1/L2），生图指令自动加 `red_safe_block`。

## 成语 / 趣学成语分支

- **与语文共用** `yuwen_lib.gen_three_cases`
- **输入**：`故事类内容-趣学成语tab新增内容.csv` 类表（课文标题、正文、寓意、一句话介绍）
- **学名**：成语名（无书名号、无「成语《》」前缀）——以用户最新确认为准
- **标签**：固定 `千问小讲堂`
- **构建**：`build_chengyu.py` → `outputs/趣学成语-封面文案-3cases.xlsx`

## 尚未单独脚本化的学科

英语、科学等：**默认复用数学工作链路**（用户原需求）。  
新开学科时：复制 `templates/新学科需求登记表.md`，在 `skills-引用/` 增加一篇摘要，再让 Cursor 仿 `cover_lib` 或 `yuwen_lib` 新建 `*_lib.py`。
