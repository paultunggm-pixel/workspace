# 初中数学 · 纯生图配图 Skill（链路 A-仅配图）

> **适用**：初一～初三知识点配图；**只产出 A/B/C 生图指令**，不含钩子、学名、标签、徽标、压字。  
> **画风**：小高 Swiss 2.5D 等距 + 构图铁律；**底色改为低饱和浅色纯色**（非小高猩红/宝蓝高饱和）。  
> **代码**：`mentorx…/scripts/junior_math_lib.py` + `gen_junior_math_prompts.py`

---

## 1. 与小高的差异

| 项 | 小高数学（封面一体化） | 初中数学（本 Skill） |
|----|------------------------|----------------------|
| 产出 | 钩子+标签+学名+生图 | **仅 3 条生图指令** |
| 背景 | 高饱和纯色 #FF3B30 等 | **浅色 pastel** #E3F2FD、#E8F5E9 等 |
| 点缀色 | 强对比 accent | 中等饱和，仅点亮「被激活」角/边/块 |
| 构图 | 上65%左40%空，右下35% | **同** |
| 禁忌 | 无汉字 | **同** + 无标题区/徽标暗示 |

---

## 2. 固定画风（Agent 必遵守）

### 构图（与小高一致）

- 3:4 竖版；画布**铺满**浅色纯色底，四边无白边  
- **上方 65%、左侧 40% 绝对空白**  
- 主体压缩在**右下** ≤35% 高 ×60% 宽，可溢出右/下边缘裁切  
- 禁止：竖直绳/杆、内框、对角装饰线、画面上半区任何元素  

### 色彩（初中专用）

- **背景**：低饱和浅色纯色（雾蓝、薄荷、浅杏、藕荷、云灰蓝、奶油黄…）  
- **主体**：#F0F0F0 灰白 + 粗黑轮廓  
- **点缀**：每案 1 个 accent，**仅**「被工具激活」的角/边/块变色  
- 禁止第四色相、霓虹、渐变、Pixar 质感、拟人五官  

### 叙事公式（与小高一致）

**数学主体（灰白）+ 介入工具（从右/下伸入）+ 单一动作定格 + 局部激活变 accent**

主体/工具/动作必须一眼能联想到**本课知识点**（见底表「课文标题」「知识点关键词」「知识点说明」）。

---

## 3. 输入表（生产运营底表）

必填列：`课文标题`、`知识点关键词`、`知识点说明`、`知识点类型`、`学段`  
可选：`大章节名称`、`小章节名称`、`教材节数`、`核心教学目标`

**不要**求钩子/学名/url 列。

---

## 4. 3-case 轮转规则

| Case | 差异 |
|------|------|
| **A** | 配色组 1 + 三角尺 + 冷色日光 |
| **B** | 配色组 2 + 圆规 + 暖色日光 |
| **C** | 配色组 3 + 量角器 + 中性教室光 |

配色由 `课文标题+关键词+学段` 哈希从 8 组浅色对中轮转，**同课 A/B/C 底色各不相同**。
