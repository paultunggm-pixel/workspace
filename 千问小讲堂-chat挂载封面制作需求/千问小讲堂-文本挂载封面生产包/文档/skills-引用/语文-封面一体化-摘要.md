# 语文 · 封面一体化创作（摘要）

> 完整 skill 见用户 Downloads 或 `../refs/`；实现见 `scripts/yuwen_lib.py`、`YUWEN_AUTHOR_SPEC.md`。

## 画风

**电影级中国风数字插画**：水粉块面、半写实、单光源戏剧光；3:4 竖版；**画面零文字**。

## 构图铁律（v15g/v16）

- 主角**全身**在画布**上半部**，脚约纵向 **50%**，全身高 ≤60%
- **下半部**留空：地面同色延续，纹理向下稀释，无主体无字
- 主角偏**中/右**，**左上角**仅天空/远景
- 背景大色块；主角精致写实

## 文案

每条 **3 个钩子**（≤12 字）：

1. 悬念反问  2. 核心动作  3. 情感共鸣  

**标签**：固定 `千问小讲堂`  
**学名**：`[类型]《篇目》`（篇目 ≤6 字）— **用户可改**（成语 tab 仅成语名）

## 创作顺序

1. **考据** kaoju（朝代、服饰、道具、场景）  
2. titles ×3 + summary ≤15  
3. hero / scene / ground / light / mood / footing  
4. `render_yuwen_prompt` 自动拼接铁律 + 英文锚句  

## 3-case 轮转

- 共享考据与画面描述  
- A/B/C 换三钩子 + 色调/光照变体（主调 / 暖黄昏 / 清透晨光；夜戏只在冷暖月光间变）

## 红色主题

`detect_red_theme` → L1（领导人：无人物空镜）/ L2（先烈：背影剪影）  
生图自动加 `red_safe_block`。

## 禁止

- prompt 里写「给标题留白」「text overlay」  
- hero 里写牌匾对联招牌题字  
- 功利引导、课堂腔、画面汉字  

## 脚本

`yuwen_data_*.py` → `yuwen_batch.py` → `build_yuwen.py` → `validate_yuwen.py`
