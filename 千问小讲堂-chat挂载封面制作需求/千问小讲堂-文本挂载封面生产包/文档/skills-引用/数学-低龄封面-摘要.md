# 数学 · 低龄封面（摘要）

> 完整：`../refs/千问小讲堂·低龄数学_封面一体化创作.md`  
> 代码：`scripts/cover_lib.py`（PIXAR 模板）

## 适用

学龄前 ~ **三年级**；3:4 竖版。

## 画风

- **Pixar / Illumination** 级 3D 硅胶玩具质感  
- 明亮**纯色背景**（9 色池轮转）  
- 角色群在**右下**，最高点 ≤ 画面高 40%，**上半留白**  
- 主角 1 个，体型 2–3 倍于配角，总角色 ≤5  

## 文案

- **钩子**：三公式，口语化  
- **标签**：`【年级】★【关键词】`  
- **卡片标题**：课程短名压缩 ≤8 字  
- **生图**：`PIXAR_PREAMBLE` + 角色描述 + `PIXAR_TAIL`  

## 生图禁忌（v2）

- **严禁画面任何汉字/字母/数字**（英文负向锚）  
- 不要写「上方留白给文字」类正向描述  

## 3-case

配色 / accent / 钩子公式轮转；`batch_gen.py` + `build_v2.py`
