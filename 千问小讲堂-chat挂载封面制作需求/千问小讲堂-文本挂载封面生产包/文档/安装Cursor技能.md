# 安装 Cursor 技能（一次性，可选）

让 Cursor 在你提到「封面」「3-case」「压图」时自动读本启动包。

## 方法 A：手动复制（推荐）

1. 复制本文件夹内的 `SKILL.md`
2. 粘贴到项目根目录下：

```
.cursor/skills/qianwen-cover-production/SKILL.md
```

（若没有 `qianwen-cover-production` 文件夹，新建即可。）

3. 重启 Cursor 或新开对话，说：「按千问封面启动包做 XX 学科」

## 方法 B：在对话里 @ 文件

不安装也行。每次开工 `@千问小讲堂-封面生产启动包/00-从这里开始.md` 即可。

## 方法 C：用户级技能（所有项目可用）

复制 `SKILL.md` 到：

```
~/.cursor/skills/qianwen-cover-production/SKILL.md
```
