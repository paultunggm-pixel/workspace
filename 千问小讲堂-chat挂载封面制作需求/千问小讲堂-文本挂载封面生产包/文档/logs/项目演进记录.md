## 已知坑（勿重复）

| 坑 | 正确做法 |
|----|----------|
| 满铺原图 + 2 行大字 → 必压主体 | 底图 normalize 或出图阶段主体下移 45% |
| 纯色补边 | 边缘像素延展 |
| prompt 写「留白给标题」 | 只用 NOTEXT 负向，正文不提文字区 |
| 6 字钩子拆 2 行 | `layout_low_figma` 优先 1 行 |
| 成语学名加书名号前缀 | 挂载压图：成语名本身；3-case 文案同左 |
| 科普学名带「文史趣闻《》」前缀 | 压图只显示《》内，前缀不上封面 |
| 用 mentorx `refs/render_covers.py` 跑 style3 | 必须用桌面工程新版（约 900+ 行） |
