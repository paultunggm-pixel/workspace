# inputs / 输入区

把待生产的源表放在这里（或 `@` 引用 Downloads 路径均可）。

## 建议命名

| 文件名示例 | 用途 |
|------------|------|
| `主表-0528.xlsx` | 全站课程清单 |
| `趣学成语-新增.csv` | 成语 tab 新增 |
| `数学-需合成.xlsx` | 已有底图 url，只要压图 |

## 最小字段

- **文案阶段**：年级、课程名/标题、一句话介绍（有则更好：内容类型、正文）  
- **压图阶段**：年级、钩子、标签、封面 url  

填好 `templates/新学科需求登记表.md` 后交给 Cursor。
