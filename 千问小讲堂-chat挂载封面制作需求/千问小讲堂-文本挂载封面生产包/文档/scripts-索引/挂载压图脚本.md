# 挂载压图脚本索引（链路 B）

> **工程根目录（真源）**：`~/Desktop/qianwen-cover-generator/`  
> **备份副本**：`output/cover-generator/_qcg/qianwen-cover-generator/`（与启动包同仓库，定期同步）

```bash
cd ~/Desktop/qianwen-cover-generator
pip3 install -r requirements.txt --user
```

---

## 核心

| 文件 | 作用 |
|------|------|
| `render_covers.py` | 主渲染：style1/2/3、成语双行、底图下载、JPG 压缩、OSS 上传 |
| `requirements.txt` | Pillow 等依赖 |
| `assets/HYXinRenWenSong65W.ttf` | style3 / 成语标题 |
| `assets/HYXinRenWenSongW.ttf` | 成语钩子 |
| `封面生成工作流与需求说明.md` | 产品向流程与卡点 |

---

## 批量入口（按学科选）

| 脚本 | 何时跑 | 输入列 | 样式 |
|------|--------|--------|------|
| `run_kepu.py` | 科普 + 妙题高招确认版 | `学名·确认版`、`url·确认版` | style3，仅《》内学名，**无钩子** |
| `run_style3.py` | 语文课文等单名 | 学名/url 确认版（列名可带·） | style3 |
| `run_chengyu_all.py` | 成语 50 篇 | 学名 + 钩子 + url 确认版 | 成语双行 + 设计成品兜底 |
| `run_yuwen_fix.py` | 语文问题封面重修 | 问题表 xlsx | style3 |

### 示例

```bash
# 科普/妙题（105 条典型）
python3 run_kepu.py \
  --csv "$HOME/Downloads/科普+妙题高招线上内容汇总.csv" \
  --out-dir output/kepu

# 语文全量
python3 run_style3.py --csv "/path/语文.csv" --out-dir output/style3

# 成语
python3 run_chengyu_all.py --csv "/path/成语.csv" --out-dir output/chengyu

# 调试
python3 run_kepu.py --csv "..." --out-dir output/kepu --limit 3
```

## 与 mentorx `scripts/` 的关系

| mentorx 脚本 | 链路 | 勿混用 |
|--------------|------|--------|
| `compose_math.py` | A · 数学压字 | 依赖旧 `refs/render_covers.py` |
| `build_yuwen.py` | A · 语文 3-case 文案 | 不产出挂载 URL |
| `run_kepu.py`（桌面） | B · 挂载封面 | 用桌面 `render_covers.py` |

新开挂载学科：优先仿 `run_kepu.py` 或 `run_style3.py` 写 `run_<学科>.py`，规则写入 `skills-引用/挂载压图-style3-摘要.md`。
