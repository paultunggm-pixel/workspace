# 输出列说明（合并进 Excel 时）

## 标准 12 列（A/B/C × 4）

在**原表所有列之后**追加，顺序固定：

| 列序 | 列名 | 含义 |
|------|------|------|
| +1 | A钩子 | Case A 封面主标题 |
| +2 | A标签 | Case A 标签（数学为年级★词；语文/成语多为「千问小讲堂」） |
| +3 | A学名 | Case A 卡片学名 / 挂载名 |
| +4 | A生图指令 | Case A 完整生图 prompt（一整段） |
| +5–8 | B钩子 … B生图指令 | 同上 |
| +9–12 | C钩子 … C生图指令 | 同上 |

> 数学历史版本可能还有「卡片标题」列在钩子与标签之间，以当次 `build_*.py` 为准。  
> 语文/成语当前 build 脚本：**钩子、标签、学名、生图** 四元组 × 3。

## 行底色（可选）

- 语文/成语合并进主表时：case 区常标**粉色** `FCE4EC`，便于与数学区分。
- 趣学成语独立表：case 区同样粉色。

## 生图同事怎么用

1. 按课程选 A 或 B 或 C 其中一列「生图指令」  
2. 复制整格到生图工具（不要拆段）  
3. 出图命名建议：`{年级}_{序号}_{case}.jpg`  
4. 回写「封面 url」列供压图阶段使用  

## 压图阶段表头（数学已用）

除上述外，压图输入表通常需要：

| 列 | 说明 |
|----|------|
| 年级 | 决定钩子字体（小低/小高） |
| 钩子 | 已选定的 case 标题 |
| 标签 | 已选定的 case 标签 |
| 封面 url / 底图路径 | AI 出图地址或本地缓存 |

输出：合成 png 路径清单。

---

## 挂载压图回写列（链路 B）

在**原表所有列之后**追加（或脚本自动补列）：

| 列名 | 格式 | 说明 |
|------|------|------|
| 封面压图压缩前url | PNG CDN | 无损，约 250–420KB |
| 封面压图压缩后url | JPG CDN | ≤50KB，线上挂载常用 |

输入必填（列名可带中间点 `·`）：

| 列名 | 链路 B |
|------|--------|
| 学名·确认版 | ✓ |
| url·确认版 | ✓ |
| 钩子·确认版 | 仅成语/需双行时 |

科普/妙题：**不要钩子**；学名压图时只取 `《》` 内文字（见 `挂载压图-style3-摘要.md`）。
