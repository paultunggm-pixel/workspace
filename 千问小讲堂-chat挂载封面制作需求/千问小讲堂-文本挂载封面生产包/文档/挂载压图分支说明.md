# 挂载压图分支说明（链路 B）

> **给谁用**：内容/运营，表上已有「确认版底图 url + 学名（+ 部分要钩子）」，要批量出**线上挂载封面**并回写 CDN。  
> **代码真源**：`~/Desktop/qianwen-cover-generator/`（本仓库 `refs/_qcg/` 为同步备份，以桌面为准）。

---

## 和「文案 3-case」链路的区别

| | 链路 A（本启动包默认） | 链路 B（本文） |
|--|----------------------|----------------|
| 输入 | 课程清单，要产 A/B/C 文案 | **学名·确认版** + **url·确认版**（+ 钩子·确认版） |
| 中间 | 生图指令 → 人工出底图 | 底图 url 已确认 |
| 输出 | xlsx 多 12 列 | **封面压图压缩前url** + **封面压图压缩后url** |
| 主脚本 | `build_yuwen.py` / `compose_math.py` | `run_style3.py` / `run_kepu.py` / `run_chengyu_all.py` |

两条链路可先后发生：先 A 出底图 → 内容确认 url → 再 B 压挂载封面。

## 表头约定（确认版）

| 列名 | 必填 | 说明 |
|------|------|------|
| `学名·确认版` 或 `学名确认版` | ✓ | 科普/妙题多为 `文史趣闻《xxx》`；压图时**只显示《》内** |
| `url·确认版` 或 `url确认版` | ✓ | 干净底图 CDN 地址 |
| `钩子·确认版` | 成语必填；科普/妙题**不要** | style3 单名模式忽略钩子 |
| `封面压图压缩前url` | 脚本回写 | PNG 无损 |
| `封面压图压缩后url` | 脚本回写 | JPG ≤50KB |

列名变体脚本已兼容（中间点 `·` / 无点）。

## 标准命令（复制到终端）

进入桌面工程：

```bash
cd ~/Desktop/qianwen-cover-generator
pip3 install -r requirements.txt --user
```

### 科普 + 妙题高招（无钩子，书名号内学名）

```bash
python3 run_kepu.py \
  --csv "/Users/xiyunmeng/Downloads/科普+妙题高招线上内容汇总.csv" \
  --out-dir output/kepu
```

### 语文全量（仅学名，无书名号则整段显示）

```bash
python3 run_style3.py \
  --csv "/path/to/语文类内容汇总.csv" \
  --out-dir output/style3
```

### 成语 50 篇（学名 + 钩子；20 张设计成品 + 30 张自动生成）

```bash
python3 run_chengyu_all.py \
  --csv "/path/to/成语表.csv" \
  --out-dir output/chengyu
```

调试可加 `--limit 3` 只跑前几条。
