# 千问小讲堂 · 课程封面生产启动包

> **给谁用**：内容/运营同学（非技术）。  
> **干什么**：批量产出「钩子 + 标签 + 学名 + 生图指令」各 3 套（A/B/C），以及（数学类）底图压字合成。  
> **怎么用**：把本文件夹路径发给 Cursor，说一句开场话术即可（见 `发给Cursor的开场话术.md`）。

---

## 一、两条链路（先选一条）

| 链路 | 你要干什么 | 看哪份说明 |
|------|------------|------------|
| **A · 文案 + 生图指令** | 还没有底图，要产 A/B/C 钩子、标签、学名、生图指令 | 下文「四步」+ `学科分支说明.md` |
| **B · 挂载压图** | 已有 **学名·确认版 + url·确认版**，要线上封面 + 回写 CDN | **`挂载压图分支说明.md`**（本包 **`脚本/`** 目录） |

两条可连着做：先 A 出底图 → 内容确认 url → 再 B 压挂载封面。

### 链路 A：文案 + 生图（4 步）

| 步骤 | 你负责 | Cursor/脚本负责 | 产出 |
|------|--------|-----------------|------|
| **① 清单** | 提供 Excel/CSV（学科、年级、课程名、一句话介绍…） | 读懂表头、对齐字段 | 待生产清单 |
| **② 文案+生图指令** | 确认画风方向（数学/语文/成语…） | 按 Skill 写 A/B/C 四套字段 | 带 12 列 case 的 xlsx |
| **③ 出图** | 把「生图指令」交给生图同事/模型，选图 | 可协助批量 URL、命名规范 | 底图 jpg/png |
| **④ 数学压字（可选）** | 提供「需合成表」（钩子、标签、底图 url） | `compose_math.py` | 数学挂载 png |

### 链路 B：挂载压图（3 步）

| 步骤 | 你负责 | Cursor/脚本负责 | 产出 |
|------|--------|-----------------|------|
| **① 确认版表** | `学名·确认版`、`url·确认版`（成语还要钩子） | 检查列名、条数 | CSV/xlsx |
| **② 压图+上传** | 确认科普只压《》内篇目、不要钩子 | `run_kepu.py` / `run_style3.py` 等 | PNG + JPG |
| **③ 回写** | 把两列 URL 交给线上 | 脚本自动回填 | `封面压图压缩前/后url` |

**你通常**：链路 A 做 ①③ 决策；链路 B 只确认表和样例，其余交给 Cursor。

## 二、按学科选哪条路（30 秒判断）

| 学科/内容 | 画风 | 学名格式 | 标签 | 主要脚本/规范 |
|-----------|------|----------|------|----------------|
| **低龄数学** 1–3 年级 | Pixar 硅胶 3D | 课程短名 | 年级★关键词 | `cover_lib.py` + `batch_gen.py` |
| **高年级数学** 4–6 年级 | 瑞士国际 2.5D | 课程短名 | 年级｜关键词 | 同上 |
| **语文** 古诗词/课文/写作/科普 | 电影级中国风插画 | `[类型]《篇目》` 或仅篇目名 | 千问小讲堂 | `yuwen_lib.py` + `yuwen_data_*.py` |
| **趣学成语 / 故事成语** | 同语文中国风 | **成语名本身**（如 `守株待兔`） | 千问小讲堂 | `build_chengyu.py` |
| **数学压字** | 不重新生图，在底图上压钩子/标签 | — | 见 `字体规则.md` | `compose_math.py`（链路 A） |
| **挂载压图** | 已有确认版底图 url | 仅《》内或整段学名 | 科普无钩子 | **`挂载压图分支说明.md`**（链路 B） |

详细分支见 **`学科分支说明.md`** + **`挂载压图分支说明.md`**。

## 四、依赖（只需装一次）

```bash
cd 脚本
pip3 install -r requirements.txt --user
```

字体已在 **`脚本/assets/`**。数学压图字体规范见 **`文档/字体规则.md`**（链路 A）。

---

## 五、推荐工作节奏（新学科第一次）

1. 填写 **`文档/templates/新学科需求登记表.md`**
2. 把源表放进 **`输入表/`**
3. 打开 Cursor，粘贴 **`发给Cursor的开场话术.md`** 里对应模板
4. 先要 **3 条样例**（小低/小高/语文各一）→ 你确认画风和钩子风格
5. 再 **全量 50/200/500 条** → Cursor 自检后给 xlsx
6. 生图同事出图 → 如需压图，再走数学/语文压图脚本
