# 发给 Cursor 的开场话术（复制即用）

把下面某一段 **整段复制** 到 Cursor 对话框，并把 `@` 文件换成你的实际路径。

---

## 通用（让 AI 先读启动包）

```
请按项目里的「千问小讲堂-封面生产启动包」执行：
1. 先读 00-从这里开始.md 和 SKILL.md
2. 判定学科分支后读 skills-引用/ 对应摘要
3. 我的需求如下：[在这里写学科、条数、源表路径、学名格式等特殊要求]
4. 先做 3 条样例给我确认，再全量；全量后自检，问题数必须为 0 或逐条列出
```

---

## 数学 · 文案 + 生图指令（3-case）

```
@千问小讲堂-线上内容汇总-0528.xlsx（或你的表路径）

按封面生产启动包·数学分支，为 [低年级/高年级/全部] 数学课生成 A/B/C 三套：
钩子、标签、卡片标题、生图指令。写入 xlsx 追加 12 列。
源表以我的表为准，不要改样例池。先给 3 条样例（1 条小低 + 1 条小高 + 1 条边界），确认后全量并 validate。
```

## 数学 · 底图压字合成

```
先做 5 张小低样例，我确认后再全量。
```
