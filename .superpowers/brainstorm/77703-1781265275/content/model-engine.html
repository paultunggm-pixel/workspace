🔄 与 cc switch 的数据同步机制

数据项 来源 同步方向
当前模型 读取 ~/.claude/settings.json → model 字段 双向：App 切换 ⇄ cc switch
可用模型列表 读取 ~/.claude/models.json 或 settings 中 providers 配置 单向：cc → App
API 余额 DeepSeek API / 读取 provider usage 端点 单向：API → App（每小时）
Token 用量 Claude Code session 实时统计 单向：cc session → App

当检测到未配置任何模型时的引导卡片（替代模型选择器）

尚未配置模型

检测到你尚未通过 Claude Code 配置过模型提供商。
运行以下命令快速开始，或在 App 中手动配置：

终端
$ cc switch
→ 选择模型提供商（DeepSeek / Anthropic / OpenAI 兼容）
→ 输入 API Key（存入 macOS Keychain 🔐）
→ 重启 App 后自动识别
App 不存储密钥 · 配置由 Claude Code 统一管理

📊 数据流全景

┌─────────────────────────────────────────────────┐
│              数据流向                              │
│                                                    │
│  ~/.claude/settings.json ──→ App 读取当前模型       │
│  ~/.claude/models.json   ──→ App 读取可用模型列表    │
│  cc switch                ←→ App 双向同步模型选择   │
│  platform.deepseek.com   ──→ App 定时拉取余额       │
│  Claude Code session     ──→ App 实时 Token 统计    │
│                                                    │
│  🔐 密钥全部由 cc 管理，App 不接触                  │
└─────────────────────────────────────────────────┘
